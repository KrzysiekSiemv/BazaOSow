Windows 7

Pamięć RAM 1 GB dla wersji 32-bitowej lub 2 GB dla wersji 64-bitowej
Miejsce na dysku twardym 16 GB dla 32-bitowego lub 20 GB dla 64-bitowego systemu operacyjnego
Karta graficzna biblioteki DirectX 9 lub nowsze ze sterownikiem WDDM
Wyświetlacz 800 x 600

Windows Server 2008
